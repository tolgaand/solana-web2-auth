## Run Locally

Clone the project

```bash
  git clone https://github.com/tolgaand/solana-web2-auth
```

Go to the project directory

```bash
  cd backend/workers/auth
```

Install dependencies

```bash
  yarn install
```

Run Tests

```bash
  yarn test
```

Start the server

```bash
  yarn dev
```

Deploy to cloudflare

```bash
  yarn publish
```
